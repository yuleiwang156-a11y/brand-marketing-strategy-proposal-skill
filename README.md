# 品牌营销战略建议书生成 Skill

这是一个用于生成品牌营销战略建议书的 Agent Skill。

## 适用场景

用于根据客户资料，完成需求理解、问题定义、研究分析、机会点洞察、目录搭建、逐页 PPT 内容脚本生成和 Word 文档输出。

## 核心流程

客户资料
→ 确认点 1：客户需求理解与项目问题定义
→ 研究分析层
→ 确认点 2：具体机会点洞察
→ 确认点 3：客户可见目录与内部分析结构
→ 逐页生成 PPT 内容脚本
→ 质量检查
→ 生成 Word 文档

## 使用方式

将客户资料放入：

```text
inputs/客户资料/客户名称/
```

然后在 Agent 中输入：

```text
请按照本项目 Skill 执行。本次客户资料在 inputs/客户资料/客户名称/ 中，请从确认点 1 开始。
```

## 注意事项

不要把客户资料上传到 GitHub。
不要把 MEMORY.md、agent_memory、outputs、backups 上传到 GitHub。
